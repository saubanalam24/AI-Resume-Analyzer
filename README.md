# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer built using Flask, Python, NLP, PDF processing, and data visualization.

This application analyzes resumes against job descriptions, calculates match scores, identifies missing skills, generates charts, and creates downloadable PDF reports.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/saubanalam24/AI-Resume-Analyzer.git
```

---

## Go to Project Folder

```bash
cd AI-Resume-Analyzer
```

---

## Create Virtual Environment

```bash
python -m venv venv
```

---

## Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / Mac

```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Flask App

```bash
python app.py
```
